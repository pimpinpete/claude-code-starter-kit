# Claude Code Starter Kit

34 specialist agents, 24 plugins, optimized settings — one command installs everything.

## Install

Open Terminal and run:

```bash
curl -fsSL https://raw.githubusercontent.com/pimpinpete/claude-code-starter-kit/main/install.sh | bash
```

Then restart Claude Code. Done.
